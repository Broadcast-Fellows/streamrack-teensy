# streamrack-teensy

Proof-of-concept of Companion-Satellite rebuilt for Teensy 4.1, which has an ARM Cortex-M7 at 600 MHz and a built-in Ethernet transciever. Only compatible with Stream Deck MK.2 (12-key).
